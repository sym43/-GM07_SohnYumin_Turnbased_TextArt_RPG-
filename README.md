- 프로젝트 이름 : Turnbased_TextArt_RPG
- 개발언어 : <img src="https://img.shields.io/badge/c%23-%23239120.svg?style=for-the-badge&logo=csharp&logoColor=white">
## 게임 설명
- 플레이어는 맵을 돌아다니며 상호작용 할 수 있다.
- 장애물 및 적 오브젝트는 통과할 수 없고,
포탈 오브젝트는 통과할 수 있다.
- 포탈은 위에 올라가서 상호작용을 누르면 작동되며,
적은 근처 1칸 상하좌우에 있을 때, 상호작용을 누르면 전투를 시작한다.

- 전투는 적의 다음 행동을 미리 보여주고, 플레이어가 행동을 선택하면,
턴이 진행되며 행동을 주고 받는 슬더스와 비슷한 턴제 전투이다.
(카드 시스템은 없음.)
